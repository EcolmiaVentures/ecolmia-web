ECOLMIA VENTURES — Web (Versión 2)

Cómo actualizar el repositorio en GitHub:
1) Descarga TODOS los ECOLMIA_v2_part*.zip
2) Descomprime cada ZIP en la misma carpeta local del proyecto (se combinarán).
3) Sube TODO el contenido al repo (Code → Add file → Upload files → arrastra carpetas y archivos).
4) Commit changes.
5) Espera 1–2 min y revisa en https://ecolmiaventures.github.io/ecolmia-web/

Notas:
- El menú superior es común y se inyecta con scripts/header.js
- Puedes cambiar los placeholders de imágenes en assets/img/
- Favicon/logo editables en assets/favicon.svg y assets/logo.svg
- 404.html, robots.txt y sitemap.xml ya están incluidos.
